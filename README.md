# HOW TO RUN
1. Download git files
2. change directory to the /covidmap folder
3. Run $ npm install
4. Run $ npm start
5. after npm install, only npm start is needed to run the site

# Services Used
- ReactJS
- Bootstrap 
- Domo embedded graphs
